# Jessica Financial OS - Template

Personal Treasury Dashboard 教學範本

## 快速開始

```bash
# 1. 安裝依賴
npm install

# 2. 啟動開發環境
npm run dev

# 3. 打開瀏覽器
http://localhost:5173
```

## 資料更新

編輯 `data/records.json` 來更新財務數據。

## 部署到 GitHub Pages

1. Fork 這個 repo
2. Settings → Pages → Source 選 "GitHub Actions"
3. 推送到 main 分支會自動部署
